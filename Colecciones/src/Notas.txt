
Las Collections
============================================================================================================================
// Antes de profundizar en las Collections, debemos entender qué es el Java Collections Framework.
//--Java Collections Framework:
// Es todo un conjunto de clases, interfaces y algoritmos que java provee
// para trabajar con Colecciones de datos de forma estándar, no es una sola clase ni interface es toda
// una estructura estándar.

//--Colecciones en Java:
// una colección es un conjunto de elementos que permite
// almacenar, recorrer y manipular, pudiendo o no tener orden dependiendo del tipo.
// Por ejemplo, podemos imaginarlo como una carpeta de correos
// que contiene múltiples mensajes.

// En Java, algunas colecciones forman parte de la jerarquía cuya raíz es la interface Collection

//--Collection:
// Es la interface raíz de la jerarquía de colecciones,
// definiendo el comportamiento fundamental para agrupar objetos individuales (elementos) en una sola unidad.
// public interface Collection<E> extends Iterable<E> {},
// esta interface proporciona métodos básicos cómo por ejemplo
// boolean contains(Object 0):
// Este metodo retorna true si y solo si la colección contiene un elemento especificado
// int size():
// Este metodo retorna el número de elementos de la colección
// boolean isEmpty():
// Este metodo retorna true si la colección NO contiene elementos
// boolean add(): permite agregar un nuevo elemento a la colección Si esta colección cambió como resultado de la llamada
// boolean remove(): permite eliminar un elemento de la colección retorna true si el elemento se removió correctamente.
// toArray(): Devuelve un arreglo (Object[]) que contiene todos los elementos de la colección.



// A partir de Java 21, algunas colecciones que mantienen un orden definido
// extienden la interfaz SequencedCollection, la cual introduce métodos como:

// addFirst(): agrega un elemento al inicio
// addLast(): agrega un elemento al final
// getFirst(): obtiene el primer elemento
// getLast(): obtiene el último elemento
